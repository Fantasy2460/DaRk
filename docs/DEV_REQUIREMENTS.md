## 2. 核心玩法机制

### 2.1 核心循环
描述玩家在一局游戏中的主要行为循环：
- **探险阶段**：从主城进入黑暗森林 → 在区域内清理怪物 → 搜刮掉落的装备/道具 → 在区域内即时穿戴提升战力
- **抉择阶段**：清理完成后选择「层层深入」进入下一层更危险的区域，或「退出冒险」安全返回主城
- **结算阶段**：存活状态下退出，背包和身上的装备带回主城；死亡则丢失本次探险获得的全部物品（局外解锁内容除外）
- **养成阶段**：在主城区穿戴/整理装备，后续开放强化、宝石镶嵌等功能

### 2.2 Roguelike 要素
- [x] **永久死亡**：角色死亡后丢失本次探险获得的全部物品（局外解锁内容除外）
- [x] **随机地图生成**：森林大地图随机刷怪，非房间制；伪随机种子 [TBD]
- [x] **随机道具/技能**：击败怪物概率掉落装备/道具
- [x] **局外成长（Meta-progression）**：无角色解锁（开局全职业可选）；天赋树进度局外永久保留（需要持久化存档）
- [x] **局内成长（Run-progression）**：局内拾取并即时穿戴装备提升战力，层层深入难度递增

### 2.3 胜利与失败条件
- **胜利条件**：找到传送门并主动选择撤离，存活返回主城
- **失败条件**：生命归零（死亡后丢失本次探险获得的全部物品）
- **特殊机制 - 迷雾值**：随时间增加，玩家可见度越来越低，增加探险压力和风险决策

## 5. 地牢与关卡设计

### 5.1 地图结构
- **层数/章节数量**：线性层层深入，每层一个森林大地图
- **地图类型**：非房间制，开放式森林大地图，渲染尺寸较大，支持自由探索
- **地图连接方式**：线性层层深入，清理当前层后可选择进入下一层或撤离
- **小地图是否显示**：[TBD]
- **传送门**：每层随机位置生成传送门，玩家找到后可选择撤离或继续深入

### 5.2 怪物与 AI
- **刷怪机制**：森林地图内随机刷怪，怪物不会主动全图寻敌
- **仇恨机制**：玩家进入怪物一定范围后触发仇恨，怪物开始追击/攻击
- **AI 复杂度**：基础追击 + 攻击，远程怪可保持距离

## 6. 角色系统

### 6.1 玩家角色
- **可选职业**：3种，开局即全解锁（无角色解锁门槛）
  - **战士**：近身搏斗，高生命高防御，近战范围攻击
  - **法师**：远程范围输出，高爆发，低生命，依赖法力值
  - **贤者**：增益/治疗/减益，辅助控制，团队增益（后续可拓展组队）
- **职业差异化**：每个职业拥有专属主动技能 + 专属天赋树分支
- **天赋树系统**：所有职业共享基础天赋树，玩家可选择自己的分支方向；天赋树进度局外永久保留

### 8.2 掉落与品质
- **品质分级**：C（普通）/ B（优秀）/ A（史诗）/ S（传说）四级
- **测试版属性**：只增加固定数值属性（攻击/防御/生命等），后续拓展套装效果
- 掉落是否受运气属性影响？[TBD]
- 是否有专属/传说物品？[TBD]

### 8.3 背包与场景隔离
- **局内背包**：只有从主城带入的道具才能在局内使用；局内拾取的装备即时穿戴或存入局内背包
- **主城背包**：与局内背包完全隔离，回到主城后才能整理、存取、穿戴
- **死亡惩罚**：角色死亡后，本次局内获得的所有物品（包括已穿戴）全部丢失
- **背包容量限制**：6×3 = 18 格（局内背包与主城背包各自独立，均为 18 格）
- **合成/熔炉系统**：后续版本开放装备强化、宝石镶嵌

### 8.4 主城 NPC 商店
- **药水 NPC**：出售不同级别的药水，级别越高效果越好
- **装备 NPC**：每小时自动刷新可售商品，有概率刷出 S 级高级装备；玩家可花费游戏币手动刷新商店
- **货币**：游戏币（局内获取，用于主城消费）
